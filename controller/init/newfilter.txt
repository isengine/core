	// работа фильтра
	// 
	// фильтр работает со всеми полями записи, включая id, name, parents, ctime и др., а также с полями данных
	// для полей записи нужно использовать выборку вида 'имя_поля:значение'
	// для полей данных нужно использовать выборку вида 'data:имя_поля_данных:значение'
	// по-умолчанию выборка идет по 'and', т.е. при полном совпадении, но можно переключить по 'or'
	// 
	// для формата json
	// 
	// массив со следующими группами:
	//   ключ - имя поля вида 'имя_поля' или 'data:имя_поля_данных'
	//   массив значений вида
	//     значение поля
	//     тип фильтрации
	//     тип объединения/исключения
	// например:
	//   {
	//   	"method" : "and",
	//   	"filters" : [
	//   		{
	//   			"name" : "parents",
	//   			"data" : false,
	//   			"values" : {
	//   				"name" : "one",
	//   				"type" : "string",
	//   				"require" : true,
	//   				"except" : true
	//   			}
	//   		}
	//   	]
	//   }
	// 
	// поддерживаются следующие типы фильтрации:
	//   equal - точное совпадение
	//   string - наличие подстроки в строке
	//   numeric - сравнение числа в диапазоне от min до max включительно (name должен быть массивом из двух чисел)
	//   noempty - поле не должно быть пустым
	// 
	// поддерживаются следующие типы объединений:
	//   по-умолчанию, должно присутствовать хотя бы одно значение
	//   require - это значение должно присутствовать обязательно
	//   except - это значение обязательно должно отсутствовать
	// 
	// для передачи в одну строку
	// 
	// вы указываете сколько угодно значений через двоеточие и сколько угодно полей для выборки через вертикальную черту
	// например
	//   'one:1:2|two:a:b:c'
	// если же вы хотите задать выборку всех непустых полей (т.е. всех полей, кроме пустых, и неважно чем они заполнены), используйте поле без значений
	//   'one|two:a:b:c'
	// если вы хотите использовать точное совпадение всех элементов, используйте для перебора вместо двоеточия ':' знак плюс '+'
	//   'one:1+2|two:a+b+c'
	// например, этот фильтр используется в работе контента для применения точного совпадения по родителям
	// если вы хотите, чтобы в поиске были выданы данные, у которых совпадает только один элемент из перечисленных, используйте вначале каждого значения знак плюс '+'
	//   'one:+1:+2|two:+a:+b:+c' или для одного значения 'one:+1'
	// если вы хотите, чтобы в поиске были выданы данные, у которых отсутствуют перечисленные элементы, используйте вначале каждого значения знак минус '-'
	//   'one:-1:-2|two:-a:-b:-c' или для одного значения 'one:-1'
	// если вы хотите использовать поиск строки, используйте оператор '*'
	//   'one:*search'
	// если вы хотите использовать поиск чисел в диапазоне, используйте оператор '_'
	//   'one:1_' - даст все значения, равные и больше 1
	//   'one:_10' - даст все значения, равные и меньше 10, в т.ч. отрицательные
	//   'one:1_10' - даст все значения от 1 до 10 включительно
	// данный оператор работает также
	//   с числами c десятичными долями (числа с плавающей точкой вида 0.1)
	//   с отрицательными числами
	// будьте внимательны:
	//   все сравниваемые строки будут преобразованы в число
	//   пробелы уберутся
	//   запятые автоматически преобразуются в точки
	//   точки будут считаться разделителями десятичных долей, а не разрядов
	// например:
	//   10.000 будет преобразовано в 10
	//   0,5 будет преобразовано в 0.5
	//   false будет преобразовано в 0
	//   true будет преобразовано в 1
	//   'string' будет преобразовано в 0
	// если вы хотите найти строку с нижним подчеркиванием, используйте оператор '*', т.к. он имеет приоритет:
	//   '*_'
