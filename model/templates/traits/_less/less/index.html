Less Rendering Preview

Для того, чтобы скомпилировать less в css, просто поместите в папку с этим файлом файл style.less и обновите страницу.
Готовый код будет помещен в контейнер <style> в теге <head> этой страницы!

Чтобы скомпилировать less в css, перейдите по этой ссылке: compile.php (не работает с локальными файлами)

Пример исходного кода

Как он будет скомпилирован

 
@base: #f938ab;

.box-shadow(@style, @c) when (iscolor(@c)) {
  -webkit-box-shadow: @style @c;
  box-shadow:         @style @c;
}
.box-shadow(@style, @alpha: 50%) when (isnumber(@alpha)) {
  .box-shadow(@style, rgba(0, 0, 0, @alpha));
}
.box {
  color: saturate(@base, 5%);
  border-color: lighten(@base, 30%);
  div { .box-shadow(0 0 5px, 30%) }
}

.box {
  color: #fe33ac;
  border-color: #fdcdea;
}
.box div {
  -webkit-box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
  box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
}
